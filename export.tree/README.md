# PAFIN: PennLINC AFfective INstability- fMRIPrep Derivatives

This dataset contains fMRIPrep derivatives for the PAFIN dataset (https://openneuro.org/datasets/ds006131).

These derivatives were produced by first running a Python translation of NORDIC (https://github.com/nipreps/NORDIC_Raw/tree/executable), followed by fMRIPrep 25.2.4, using BABS (https://pennlinc-babs.readthedocs.io/).

The processing code is available at https://github.com/PennLINC/affective-instability/tree/main/processing/nordic-fmriprep.
